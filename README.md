# PocketViewer
AWS S3のファイルビューアーです。

## 機能要件
- ディレクトリ/ファイル参照
- ディレクトリ/ファイルのS3アップロード

## 使用技術
- Nuxt.js(Vue.js)
- Vuetify

## Build Setup
``` bash
# create env file
# Please change the .env_dev file
$ cp .env.template .env_dev

# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev
```
